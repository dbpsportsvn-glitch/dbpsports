Upload File Nhạc

  • Chọn file: Click vào trường "Music file" để chọn file nhạc từ máy tính
  • Định dạng hỗ trợ: MP3, WAV, OGG, M4A, AAC
  • Tự động lưu: File sẽ được lưu vào thư mục playlist tự động
  • Đường dẫn: Đường dẫn file sẽ được tạo tự động, không cần nhập

Tạo Playlist Nhanh từ Thư mục

💡 Mẹo: Thay vì thêm từng bài hát một, bạn có thể tạo playlist hoàn chỉnh từ thư mục:

  1. Vào Playlists → Chọn action "Tạo playlist từ thư mục"
  2. Nhập tên playlist và đường dẫn thư mục
  3. Hệ thống sẽ tự động tạo playlist và import tất cả file nhạc
🎵 Tự động phân tách tên file: Nếu tên file có định dạng "Nghệ sĩ - Tên bài hát", hệ thống sẽ tự động điền vào các trường "Nghệ sĩ" và "Tên bài hát". Ví dụ: "Taylor Swift - Love Story.mp3"

Lưu ý quan trọng:

  • File sẽ được lưu vào thư mục của playlist đã chọn
  • Nếu file đã tồn tại: Track hiện tại sẽ được cập nhật với file mới
  • Thời lượng sẽ được tính tự động khi phát nhạc
  • Thứ tự có thể được điều chỉnh sau khi upload
  • Tên bài hát không bắt buộc - sẽ tự động lấy từ tên file
  • File trùng lặp: Hệ thống sẽ tự động xử lý và cập nhật track hiện có
